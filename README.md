# 📘 Quizoo – Web-Based Quiz Application

Quizoo is a simple and interactive **web-based quiz application** built using **HTML, CSS, and JavaScript**.  
It demonstrates basic frontend development concepts like DOM manipulation, event handling, and UI design.

---

## 🚀 Features

- ✅ Single-question quiz interface  
- ✅ Option selection system  
- ✅ Submit-based answer checking  
- ✅ Live score update  
- ✅ Visual feedback (correct ✅ / incorrect ❌)  
- ✅ Clean and modern UI  

---

## 🛠️ Tech Stack

- **HTML** – Structure  
- **CSS** – Styling  
- **JavaScript** – Logic & interactivity

## 🎮 How It Works

1. User selects an option  
2. Clicks on **Submit** button  
3. Application checks the answer:
   - ✔ Correct → Option turns green & score increases  
   - ❌ Incorrect → Option turns red & correct answer is shown  
4. Score updates on screen
